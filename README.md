
## Overview
This is a simple web application that generates QR codes from text input. Follow the steps below to use it.

## Steps to Use

### 1. **Open the Application**
   - Open index.html in your web browser

### 2. **Enter Text**
   - Click on the input field in the white bar at the top
   - Type the text, URL, or any information you want to encode into a QR code
   - The entered text will display below the input area when you click outside the field

### 3. **Generate QR Code**
   - Click the **"Generate QR Code"** button (blue button on the right)
   - A QR code image will be generated and displayed on the screen

### 4. **View Results**
   - The generated QR code will appear below the input area
   - The text you entered will be displayed in aquamarine color above the QR code
   - You can scan the QR code with any QR code reader app on your phone

### 5. **Generate Another QR Code**
   - The input field will automatically clear after generating a QR code
   - Simply repeat steps 2-4 to generate a new QR code

## Features

- ✅ Real-time text display as you enter data
- ✅ High-quality QR code generation
- ✅ Clean, modern dark theme interface
- ✅ Responsive design that works on mobile devices
- ✅ Automatic clearing of input after generation
